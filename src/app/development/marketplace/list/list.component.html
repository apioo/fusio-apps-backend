Browse and install bundles from the Fusio marketplace. A bundle contains action, schema, event, cronjob and trigger configurations which you can install to solve specific use cases.

App

Browse and install apps from the Fusio marketplace. An app is a small javascript based app which works with the Fusio API like the developer portal. By default all apps are installed under public/apps.
